투자종류선택

누구에게나 열려있는 소액 품앗이 대출 popfunding

팝펀딩은 다수의 투자회원으로부터 투자를 받아
투자 금액이 신청한 금액의 100% 달성되면, 빌릴 수 있는
소액 품앗이 대출 서비스입니다.

  • 신청자격

    만 19세 이상이라면 신용 제한 없이 누구나 가능합니다.

  • 기간

    6,9,12개월 (100~250만원)과
    15,18,21,24개월 (300만원)로 나눠어 집니다.

  • 대출회원 혜택 빌리기 금액

    100만원, 150만원, 200만원, 250만원, 300만원이며
    우수대출회원은 1000만원까지 가능합니다.
    ( KT회원 소액빌리기 : 10/20/30민원 )

  • 이자와 상환방법

    연 0~25% (자율설정)이고, 원금 균등상환입니다.
    ( KT회원 소액빌리기 : 연 9.9% 고정금리, 원금균등상환 )
