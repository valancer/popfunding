POPFUNDING 기본 용어 설명

  • 빌리기

    [ 대출 ] 이자와 기간을 정하고, 여러 투자자들에게 십시일반 대출을 받습니다.

    예) A님은 130명의 투자자들에게 300만원 / 20% / 24개월로 빌리기를 달성하셨습니다.

  • 투자하기

    [ 투자 ] 한 투자건(빌리기)에 여러 투자자들이 모여서 이자를 받고 빌려줍니다.

    예) B님은 130명의 투자자들과 함께 A님에게 300만원중 6만원을 투자 하셨습니다.

  • 개인투자

    [ 투자상품 ] 개인(신용)에게 투자하는 행위로 신청금액의 1/50로 제한 됩니다.

    예) B님은 A님이 신용으로 빌리려는 1000만원중 20만원(최대금액)을 투자합니다.

  • 기업투자

    [ 투자상품 ] 개인(법인)과 소기업의 신용 or 담보에 금액 제한 없이 투자합니다.

    예) A님은 기업투자에 담보/신용을 확인하고, 제한없이 3000만원을 투자하였습니다.

  • 동산 담보 기업투자

    [ 투자상품 ] 소상공인이 재고를 담보로 빌리기신청한 투자건입니다.

    예) A님은 창고의 신발재고 담보물을 보고, 소상공인에게 1500만원을 투자하였습니다.

  • 음악저작권 담보 기업투자

    [ 투자상품 ] 음악인이 음악저작권을 담보로 빌리기신청한 투자건입니다.

    예) A님은 한국음악저작권협회의 음원 담보물을 보고, 음악인에게 500만원을 투자하였습니다.

  • KT개인투자

    투자상품 ] KT사용자에게 빌려주는 개인투자입니다.

    예) B님은 KT회원인 A님이 빌릴려는 30만원중 6천원을 투자합니다.

  • 채권

    [ 거래상품 ] 투자가 달성되면 투자에 대한 권리로서 채권을 받고, 회원간 거래가 가능합니다.

    예) A님은 10건의 투자상품에 투자를 하고, 10개의 채권이 생겼습니다.

  • 달성

    [ 빌리기성공 / 투자성공 ] 투자금이 신청금액의 100%를 채우고 성공한 상태입니다.

    예) 이번 기업투자의 신청금액 1억원이 모두 투자되어 100%를 달성하였습니다.

  • 채권마켓

    [ 거래장소 ] 채권을 회원간에 거래하여, 현금으로 전환하는 장소입니다.

    예) A님은 채권마켓에서 보유한 채권중 5개만 매도하여 100만원의 현금을 확보 하였습니다.
    예) B님은 채권마켓에서 5개의 채권을 매수하여 15만원의 수익이 발생 하였습니다.

  • 달성실패

    [ 빌리기실패 / 투자실패 ] 투자금이 100%를 채우지 못하고, 기간이 만료된 상태입니다.

    예) 이번 기업투자의 신청금액 1억원중 90%만 투자되어 투자건이 취소되었습니다.

  • 투자자 인증 / 대출자 인증

    [ 본인인증 ] 투자자인증/대출자인증을 통한 본인확인 후 원하는 서비스를 이용할 수 있습니다.

    예) A님 투자자인증을 받지 않아, 투자를 하지 못 했습니다.

  • 연체

    [ 이자 미납입 ] 이자나 원금을 1일~30일 동안 상환하지 않은 경우입니다.

    예) A님은 이자 5만원을 상환하지 않아, 연체 상태가 되었고 개인정보가 공개 되었습니다.

  • 대손

    [ 원금 미납입 ] 이자나 원금을 31일 이상 상환하지 않은 상태입니다.

    예) A님은 지난달 이자와원금인 10만원을 31일 동안 상환하지 않아, 대손이 발생하였습니다.
